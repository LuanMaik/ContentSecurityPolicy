Carregamentos de outros domínios

Normalmente precisamos utilizar bibliotecas de terceiros, utilizando urls de CDNs externos.
O exemplo demonstra como configurar o carregamento de recursos de outros domínios.

Este arquivo faz o carregamento do bootstrap e Jquery, utilizando a CDN do cloudflare.
Abra o console do navegador e certifique-se de que não há erros de carregamento.

<meta http-equiv="Content-Security-Policy" content="default-src 'self' https://cdnjs.cloudflare.com;">

A diretiva default-src é utilizada para qualquer tipo de carregamento (js, css, img, media, etc);

O valor 'self' define que seja permitido o carregamento de dados da mesma origem (domínio) deste arquivo.

O valor 'https://cdnjs.cloudflare.com' define que seja permitido o carregamento de dados do domínio https://cdnjs.cloudflare.com.
