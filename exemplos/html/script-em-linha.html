Carregamentos de script em linha

Recursos processados em linha como javascript, também precisam ser permitidos.
Este exemplo demonstra como configurar o carregamento de javascript em linha.
Abra o console do navegador e certifique-se de que não há erros de carregamento.

<meta http-equiv="Content-Security-Policy" content="script-src 'self' 'unsafe-inline'">

A diretiva script-src é utilizada para regras de carregamento de scripts;

O valor 'self' define que seja permitido o carregamento de dados da mesma origem (domínio) deste arquivo.

O valor 'unsafe-inline' define que seja permitido o carregamento de scripts em linha.
